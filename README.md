# ModelFitness: Gym Churn Prediction 🏋️‍♂️

This project aims to predict customer churn for the ModelFitness gym chain. By leveraging Machine Learning, I identified the key factors that lead to membership cancellations and developed a customer segmentation strategy to improve retention rates and optimize marketing efforts.

## Key Technical Features
- **Predictive Modeling:** Compared **Logistic Regression** and **Random Forest** classifiers, achieving high precision in identifying at-risk members.
- **Advanced Feature Engineering:** Addressed multicollinearity and used feature importance analysis to understand the drivers behind churn.
- **Customer Segmentation:** Implemented **K-Means Clustering** and validated the optimal number of groups using the **Elbow Method**.
- **Business Impact:** Conducted a simulated economic analysis to demonstrate how targeted retention campaigns can reduce churn costs.

## Tech Stack
- **Language:** Python
- **Libraries:** Scikit-learn, Pandas, Seaborn, Matplotlib.
- **Machine Learning:** Classification (Supervised) and Clustering (Unsupervised).

# ModelFitness: Predicción de Abandono (Churn) 🏋️‍♂️

Este proyecto tiene como objetivo predecir la pérdida de clientes para la cadena de gimnasios ModelFitness. Utilizando Machine Learning, identifiqué los factores clave que llevan a la cancelación de membresías y desarrollé una estrategia de segmentación para mejorar las tasas de retención y optimizar los esfuerzos de marketing.

## Características Técnicas Clave
- **Modelado Predictivo:** Comparativa entre clasificadores de **Regresión Logística** y **Random Forest**, logrando una alta precisión en la identificación de miembros en riesgo.
- **Ingeniería de Características:** Corrección de multicolinealidad y análisis de importancia de variables para entender los detonantes del abandono.
- **Segmentación de Clientes:** Implementación de **K-Means Clustering** y validación del número óptimo de grupos mediante el **Método del Codo**.
- **Impacto de Negocio:** Simulación económica para demostrar cómo las campañas de retención dirigidas pueden reducir los costos asociados al churn.

## Herramientas Utilizadas
- **Lenguaje:** Python
- **Librerías:** Scikit-learn, Pandas, Seaborn, Matplotlib.
- **Machine Learning:** Clasificación (Supervisado) y Clustering (No supervisado).
